Base UI-like styling

Native <details> / <summary> with a Base UI-inspired layout. Header and trigger are merged into <summary>; the panel is the content that follows it.

What is Base UI?

Base UI is a library of high-quality unstyled React components for design systems and web apps.

How do I install it?

Install the package for your framework, then import the primitives you need and style them in your design system.

Can I use it with Tailwind?

Yes. Base UI ships unstyled parts, so utility classes or your own CSS can shape triggers, panels, and icons.
